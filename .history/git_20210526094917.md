# Git 分布式版本控制系统
git     分布式版本控制系统
github  Git 仓库,提供公共仓库和私有仓库
gitlab  可以在上面创建私人的免费仓库


将某个目录纳入git管理
git init(默认master分支)





工作区      --->     暂存区     --->      本地仓库    ----->    远程仓库
          git add .       git commit -m '描述'     git push
  git add <file1> <file2> ...
暂存区 ---> 工作区
git rm --cached

查看提交日志
git log 
查看文件内容
cat config 
删除已经提交的文件
git rm <file1> 
查询当前工作区所有文件的状态，以及当前处于哪个分支
git status
查看有哪些分支，带*标记即为当前所处分支
git branch -v 
创建分支
git branch [分支名]
切换到该分支  
git checkout [分支名]  

合并分支  
1.切换到接受修改的分支上  
git branch [接受修改的分支名,即被合并的分支名]
2.执行merge命令
git merge [合并的分支名]

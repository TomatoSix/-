
# setup 
1. 组件中所用到的数据、方法等等，均配置在setup中
```js
setup() {
  let name='张三'
  let age = 18

  function sayHello() {
    alert(name, age)
  }

  return {
    name, age, sayHello
  }
}
```

# ref函数
1. 添加响应式数据
```js
setup() {
  let name = ref('张三')
  let age = ref(18)

  function changeInfo() {
    name = '李四'
    age = 48
  }

  return {
    name, age, changeInfo
  }
}
```

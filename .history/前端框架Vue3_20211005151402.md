
# setup 
1. 组件中所用到的数据、方法等等，均配置在setup中
```js
setup() {
  let name='张三'
  let age = 18

  function sayHello() {
    alert(name, age)
  }

  return {
    name, age, sayHello
  }
}
```

# ref函数
1. 添加响应式数据
RefImpl(reference implement) 引用实现
2. ref 把变量'name'加工为引用对象
```js
setup() {
  let name = ref('张三')
  let age = ref(18)

  function changeInfo() {
    console.log(name)  // RefImpl {_rawValue: '张三'，value: '张三'} 对象
  }

  return {
    name, age, changeInfo
  }
}
```

# 关于vue中render: h => h(App)的解释

render: h => h(App) 是下面内容的缩写：
```
render: function (createElement) {
    return createElement(App);
}
```
进一步缩写为(ES6 语法)：
```
render (createElement) {
    return createElement(App);
}
```
再进一步缩写为：
```
render (h){
    return h(App);
}
```
按照 ES6 箭头函数的写法，就得到了：

render: h => h(App);
